## Kubernets

Repositório do curso: "Descomplicando o kubernetes" o material é completo, separado por dia de aprendizado

[https://github.com/badtuxx/descomplicandoKubernetes/](https://github.com/badtuxx/descomplicandoKubernetes/)

Estrutura básica do comando kubctl:

kubectl <comando> <recurso ou opções> <opções> --<flags>.

| Command | Description |
| --- | --- |
| minikube start | Inicia o cluster local |
| minikube stop | Para todas as máquinas locais (mantendo o último estado) |
| kubectl config get-contexts | Lista todos os contextos disponíveis |
| kubectl get pods | Obtem os pods no namespace padrão |
| kubctl cluster-info | Obtem o endereço IP externo do cluster |
| kubectl exec -it <nome do pod> <comando>. | Executa um comando dentro do pod, exemplo:kubectl exec -it mongodb-pod mongoExecuta o comando 'mongo' dentro do pod 'mongodb-pod' kubectl exec -it mongodb-pod /bin/bashAcessa a linha de comando 'bash' dentro do pod 'mongodb-pod' |
