- Pull request
  - [Git convention](https://eaton-corp.atlassian.net/wiki/spaces/CYME/pages/115218008/Git+Conventions)
    - **fix** for a bug fix
    - **feat** for implementation of a feature
    - **docs** for documentation
    - **refactor** for a refactoring
    - **chore** for a general maintenance task that doesn’t affect the code
    - Examples:
      - feat(CQAM-1234): support for super new technology
      - fix(CQAM-1235): big crash removed from feature ABC
      - docs(CQAM-1237): readme updated
  - [Commit message convention](https://eaton-corp.atlassian.net/wiki/spaces/CYME/pages/133439834/CYME+-+S+T+-+Commit+Message+Convention)
