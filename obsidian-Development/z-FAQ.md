1) How to check if a dll is `x86` or x64? (How to now the dll architecture?)
	 The tool `SigCheck` can be used to get this information, command:
	 `sigcheck.exe <<DLL path>>
	 [Download here]([Sigcheck - Sysinternals | Microsoft Learn](https://learn.microsoft.com/en-us/sysinternals/downloads/sigcheck))
