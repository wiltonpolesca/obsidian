## Suggested installation order

1) Prep VM
2) Prep a MS SQL Server installed and available
3) Download Software
4) Install PI Server
    - PI Data Archive
    - AF Server
    - PI Analysis Service
5) Install PI Web API
6) Install PI Vision
7) Import Simulation Data
8) Configure PI Vision
9) Install AF Client and SQL Client
10) Test Connections
