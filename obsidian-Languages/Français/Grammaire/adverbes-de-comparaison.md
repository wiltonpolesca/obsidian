# Quelques adverbes de comparaison

## ADVERBE + ADJECTIF + QUE

Ces termes sont employés avec l'indicatif

|       | Example                                                        |
| ----- | -------------------------------------------------------------- |
| Aussi | Cette chambre *est* **aussi grande que** la mienne             |
| Moins | Cet appartement *est* **moins lumineux que** le mien.          |
| Plus  | Les armoires *sont* **plus petites que** celles de ma cuisine. |
## ADVERBE + DETERMINANT + NOM + QUE

Ces termes sont employés avec l'indicatif

|           | Example                                                              |
| --------- | -------------------------------------------------------------------- |
| Autant    | Ce logement *a* **autant de chambres que** celui de ma soeur.        |
| Davantage | Cet appartement *a* **davantage de fenêtres que** les autres.        |
| Moins     | La propriétaire du logement *a* **moins de disponibilités que** moi. |
| Plus      | Ça te *prend* **plus de temps qu**'avant pour te rendre au travail.  |
