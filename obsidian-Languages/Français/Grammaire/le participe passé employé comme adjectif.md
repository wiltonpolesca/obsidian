# Le participe passé employé comme adjectif

Le participe passé peut être employé seul (*sans auxiliaire avoir ou être*)

---

Le participe passé employé comme adjectif s'accorde comme un adjectif (en genre et en nombre avec le nom ou le pronom qu'il accompagne).

Examples:

Le **bac** *fourni* est bleu.
- bac -> nom (m. s.)
- fourni -> p.p. (m. s.)

Plusieurs **objets** *jeté* aux poubelles sont encore utilies.
- objets -> nom (m. pl.)
- jetés -> p. p. (m. pl.)

J'ai lu sur l'**affiche** *collée* au mur dans l'entrée que les murs seront repeints vendredi
- affiche -> nom (f. s.)
- collée -> p. p. (f. s.)

Les **ordures** *éparpillées* sur le trottoir doivent être ramassées.

---

## Atention!

In ne faut pas confondre la terminaison d'un verbe conjugué avec celle du participe passé employé comme adjectif.

Exemple: Le propriétaire nous **fournit** un bac bleu. MAIS Le bac **fourni** est bleu

---

Note:

masculin: m.
féminin: f.
singulier: s.
pluiriel: pl.
participe passé: p. p.
